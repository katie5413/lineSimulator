# lineSimulator
## GBL lab project 「 超時空會客室 」

模擬跨時空的 line 聊天室

## 功能簡介

每個房間都有各自的代號，因此使用者無需註冊帳號，唯管理者在創立時需要另設密碼

這裡僅作版控

正式網址：https://myheroes.tw/minigames/LineChat/index.php

測試房號： 111111

#### 管理者

可以在後台（Backstage）設立題目、建立角色、上傳角色圖片、設計對話文本

#### 玩家

可以在會客室（Room）閱讀文本、作答


## 主要技術

前端：HTML/JS/sass

後端：php/MySQL


## 架構
